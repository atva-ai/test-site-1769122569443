# Nutrigo Natura Agrotech Private Limited - Manufacturer of Indian Spices from Guwahati - Website

This is a Next.js website generated for Nutrigo Natura Agrotech Private Limited - Manufacturer of Indian Spices from Guwahati.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
```

## Deploy

This site is configured for deployment on Netlify.
